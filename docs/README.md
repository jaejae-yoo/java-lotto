## 기능 구현 목록


1) 구입 금액 입력 
   - null, empty
   
   - 숫자가 아닌 경우
   - 1000원 단위가 아닌 경우
   - 음수인 경우


2) 로또 구매 개수 저장


3) 구매한 로또 개수 출력


4) 로또 생성 기능
   - 4-1) 1~45개의 숫자를 관리하는 기능
   - 4-2) 45개의 숫자 중 6개를 뽑는 기능


5) 당첨 번호 6개 입력
   - null, empty
   - 5-1) 당첨 번호를 “, “로 split하는 기능
       - 숫자가 아닌 경우
       - 음수인 경우
       - 숫자가 1~45 이외인 경우
   - 5-2) split한 걸 숫자로 변경


6) 보너스 볼 입력
   - null, empty
   - 숫자가 아닌 경우
   - 음수인 경우


7) 당첨 숫자와 각 로또 숫자들 비교


8) 당첨 통계 출력
